# 技术栈
Taro+Taro-Ui+React+Typescript+Stylus

## 安装项目依赖包
```
npm install
或
rm -rf node_modules
rm package-lock.json
npm cache clear --force
npm install
```

### 启动小程序
```
npm run dev:weapp
```

### 小程序云开发
> 安装wx-server-sdk(npm install --save wx-server-sdk@latest)


### 注意
```
1.css单位使用px,不想转换时使用Px/PX
```
